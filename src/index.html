QR Code Generator

A simple QR code generator service accessible via GET request.

Generating an img element

/v1/?content=Your content here

Options

Name Description
content What the qr code will return when scanned, ensure the content is url encoded
width Number of pixels width the resulting image should be (min:10, max:1000, default:100)
dark The hex code for the dark color (default: 000)
light The hex code for the dark color (default: fff)
type The output type (default: html)
  • html: An img element
  • png: A data png base64 encoded image
